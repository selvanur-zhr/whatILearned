CSS SÖZ DİZİMİ

Css kodlarımızın söz dizimi selector ve declaration kısımlarından oluşur.

Selector: Css kodumuzun etkileyeceği etiketlerin seçimini selector ile yapıyoruz.

Declaration: Css tanımlama kısmıdır.
=> Property: Özellik ismi. (örneğin ; color , background-color gibi.)
=> Value: Özelliğe verebileceğimiz değerdir. (örneğin ; blue , red gibi.)


CSS YAZIM TÜRLERİ

Css kodlarımızı 3 farklı şekilde tanımlayabiliriz. Bunlar;
- Inline
- Internal
- External

INLINE CSS
Sadece tanımlandığı etiket için geçerli olmasını istediğimiz css kodlarımızı inline olarak tanımlarız.
Örneğin ; <span style="color:red;"> text </span>

**Inline olarak tanımladığımız css kodları sadece css kodunun tanımlandığı etiketi için uygulanır ve yazı rengi kırmızı olur.

INTERNAL CSS
Internal css <head> </head> etiketleri arasına <style type="text/css"> </style> etiketleri eklenerek kullanılır.

**Internal olarak tanımlanan css kodları ise sadece içinde bulunduğu ....html sayfası içinde uygulanır.

EXTERNAL CSS
Eğer tanımladığımız css kodları birden fazla html sayfasını ilgilendiriyor ise bu durumda .css uzantılı bir dosya oluşturup bu dosya içerisine css kodlarımızı tanımlamamız gerekir.

**.css dosyası içindeki css kodlarını uygulamak istediğimiz her html sayfasına style.css dosyasını tanıtmamız gerekiyor.


ID VE CLASSES

Html etiketlerine css kodu yazabilmek için ilk önce ilgili etiketi seçmemiz gerekir. Css ile html etiketlerine 3 farklı yöntemle ulaşabiliriz. Bunlar;
- Etiket ismi
- Etiket Id
- Class
ETİKET İSMİ İLE ULAŞMAK
Bir etikete ulaşmak için etiket ismini kullandığımızda sayfadaki aynı isimdeki tüm etiketlere ulaşmış oluruz. Örnek;
p {
/* Tüm <p> etiketleri için css kodları */
}
div {
/* Tüm <div> etiketleri */
}

ID ATTRİBUTE İLE ULAŞMAK
Eğer aynı etiket ismine sahip birden fazla etiket içinden sadece bir tanesine ulaşmak istediğimizde id attribute' unu kullanabiliriz. Örnek;
<h1 id="header">
Komple Uygulamalı Web Geliştirme Eğitimi
</h1>


#header {
font-size: 32px;
font-weight:bold;
color:green;
}

CLASS İLE ULAŞMAK
Eğer belli css kodlarını birden fazla etikete vermek istediğimizde bu durumda css kodlarını bir sınıf içinde gruplarız ve bu sınıf ismini istediğimiz etikete verebiliriz dolayısıyla bu sınıfın içindeki css kodları bu sınıfa sahip olan tüm etiketlere uygulanmış olur.

Örneğin; sayfada 3 tane <li> etiketimiz var ve biz sadece ikisinin arka plan rengini yeşil yapmak istiyoruz.

.active{
background-color : green;
}


active sınıfını hangi etikete verirsek verelim o etiket yeşil arka plana sahip olur. <li> , <p> ,<div> fark etmez.

<ul>
<li class="active"> item 1 </li>
<li> item 2 </li>
<li class="active"> item 3 </li>
</ul>

<div class="active">
arka plan yeşil olur.
</div>
